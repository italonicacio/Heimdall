## Style Guide

### Arquivos

Nome de arquivos devem seguir da seguinte forma:

```bash
<nome>-<sobrenome>.hpp
<nome>-<sobrenome>.cpp
```
### Constantes

Você pode definir constante das seguintes formas.

```cpp
const <type> CONSTANT = <value>;
constexpr <type> CONSTANT = <value>;
```
